Demo webpage for the ethernet tutorial for the Simplecortex

Welcome at the demo site of the Simplecortex ethernet tutorial. This website is made to demonstrate some of the possibilities of the uIP ethernet library. If you have any questions or found a bug, please post it in our forum or email us at: Webmaster

On this site you can change a LED on the Simplecortex, read out the 2 buttons and read out an ADC pin. Every 3 seconds the page refreshes to read new values in. Showing a picture on the website is also no problem but the speed of the uIP library is not very fast, it is recomennded to keep pictures smaller then 50Kbyte

This website was the first design for the Simplecortex website. It uses HTML5 for the rounded edges. Some older browsers don't support this function. It is recommended to update your browser but the demo website functions on older browsers too.
